About MCDSS


What is MCDSS?

Malocclusion Classification Decision Support System
MCDSS is a system that used to manage dentistry patient's record and provide decision support in classifying malocclusion classes.

Who are the users?

It is used by Orthodontist, dentist and student of dentistry field which is equip with dental expertise.



About the classification

The classification engine is built using machine learning technology. The result is only a support for decision making, it is not absolute.



About Malocclusion

What is the standard used for classification?

The standard that used for the malocclusion classification is Bristish Standard Institute 1983 (BSI).

What is the input of the classification.

The input used is Cephalometric x-ray.

How many classes is involved?

Only three classes are involved. The two divisions from BSI Class II are unable to be diagnosed by using only cephalometric x-ray images.
